# Smart Job Portal System

A **Spring Boot based backend application** that provides a platform for managing job postings and user authentication. The system allows users to register, login securely using **JWT authentication**, and access job-related APIs.



## 🚀 Features

- User Registration and Login
- JWT Based Authentication
- Secure Password Encryption using BCrypt
- RESTful API Architecture
- Job Posting Management
- Exception Handling
- Database Integration using Spring Data JPA



## 🛠 Tech Stack

- **Java**
- **Spring Boot**
- **Spring Security**
- **JWT (JSON Web Token)**
- **Spring Data JPA**
- **Hibernate**
- **MySQL**
- **Maven**
- **Postman**
- **Git & GitHub**



## 🔐 Authentication Flow

1. User registers with email and password.
2. User logs in using credentials.
3. System generates a **JWT Token**.
4. Token must be included in request headers to access protected APIs.
